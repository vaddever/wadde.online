⚠ Important — 3 steps, all required:
1️⃣ Delete all existing code in the Apps Script editor, then paste in the fresh code (don't paste on top of old code).
2️⃣ Click Deploy → New deployment — not "Manage deployments → Edit". Editing an existing deployment does not push out new code.
3️⃣ Copy the new /exec URL shown after deploying — it can change — and paste it into the Script URL field above, then click Save.
Skipping step 3 is the #1 reason a redeploy "doesn't work": the portal keeps using the old URL pointing at old code. Always finish by clicking 🔍 Test Apps Script Connection below to confirm.
